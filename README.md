## What

Public go modules. Credit to Frafos GmbH.

## Use it

Import `github.com/burgesQ/gommon`.

| **package** | *description*                        |
| :-          | :-                                   |
| `assert`    | Colored assertion package            |
| `log`       | Dummy log interface & implementation |
| `pretty`    | Format and validate  json            |

### Important 

Go `1.13` is required

## Test

Simply run `make test`
